# 🚀 AI-Powered Code Documentation & Search Assistant

A smart AI tool that indexes your codebase and allows users to ask questions in plain English.

- Code search using natural language
- Extracts functions, classes, and documentation
- Semantic search using vector embeddings
- Smart product recommendation system

## 🛠 Tech Stack
- Python
- Streamlit
- Endee Vector Database
- Sentence Transformers
